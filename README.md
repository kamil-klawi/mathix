# Mathix

Mathix is a console-based scientific calculator written in C and NASM. It allows performing basic and advanced mathematical calculations.

## Technologies
- C
- NASM
- CMake
- Unity framework (unit tests)
- Docker

## Unit systems

The calculator supports three unit systems:
- [Imperial Units](https://www.britannica.com/topic/Imperial-unit)
- [Metric System](https://www.britannica.com/science/metric-system-measurement)
- Cross-system conversions between imperial and metric units (e.g., cm -> inch)
